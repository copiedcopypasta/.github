## Projects

### [Tower Defense][towerdefense] (WiSe 2025/26)
**Multiplayer Tower defense game written in Python with Pygame**
- Multiplayer
- Pixel art GUI
- Troop and tower variants

### [Wizard of OS][woos] (WiSe 2025/26)
**GreenIT website built with Next.js**
- Quest system
- Markdown-based content
- Parallax Landing Page
- Localization

### [Malefiz][malefiz] (SoSe 2025)
**CLI-based Malefiz board game written in Java**
- CLI interface
- Bot system
- Board editor
- Spectator mode
